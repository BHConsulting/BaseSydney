# BaseSydney
a Java function convert takes an integer as its argument and returns the encoded string in base 7 using the given encoding rule. 

base10 0 1 2 3 4 5 6 
base7  0 S y d n e y

Sample Input 1

7

Sample Output 1

S0

Sample Input 2

22875

Sample Output 2

Sydney

Complexity :

Order of (logN)
